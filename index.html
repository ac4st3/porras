Configuración de datos

Por defecto se consulta el JSON público de OpenFootball. No es una fuente live garantizada. Si más adelante usas una API con clave, cambia la URL o adapta la función normalizeMatches().

Muchas fuentes gratuitas no traen tarjetas rojas ni premios individuales. Por eso las rojas de Porra 2 y los premios de Porra 1 se pueden ajustar manualmente.
