Configuración de datos

Por defecto se consulta el JSON público de OpenFootball. No es una fuente live garantizada: puede actualizarse con retraso. Si usas una API con clave, cambia la URL o adapta la función normalizeMatches().

Las tarjetas rojas, mejor jugador y mejor portero pueden requerir una API deportiva más completa o carga manual.
